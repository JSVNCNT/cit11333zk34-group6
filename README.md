# Student Management System

A responsive PHP and MySQL-based Student Management System developed as a final project for CIT11333ZK34.

- User Login System
- Student Dashboard
- Subjects Management
- Grades Management
- Full CRUD Operations
- Pagination
- Bootstrap Modals

## Installation Guide

### Local Setup

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/cit11333zk34-group6.git
```

2. Move project to XAMPP `htdocs`

3. Start Apache and MySQL

4. Create database in phpMyAdmin

5. Import the SQL file

6. Configure `db.php`

7. Run the project

```text
http://localhost/cit11333zk34-group6
```
